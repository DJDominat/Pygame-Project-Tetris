Суть Проекта заключается в создании развлекательной игры Tetris при помощи языка программирования python,

написанного на языке программирования python с использованием библиотеки PyQT5

Запуск:1)Установить python версии 3.10+ (Тестировалось на версии 3.10.5)

2)Установить все библиотеки, перечисленные в requiremenets.txt файле с помощью(pip install -r requirements.txt)

3)Запустить приложение с помощью команды python main.py

Описание модулей:
1)main.py - основной код приложения, в нём реализована функция создания окна, внутригровое управление, запуск и остановку приложения, основные настройки.

2)grid.py - этот код отвечает за отрисовку поля, всё что происходит на поле, например, когда блоки полностью закрыли ряд и строка поля очищается.

3)button.py - код отвечающий за прорисовку и функциональность кнопок в главном меню

4)game.py - основной функционал программы, код отвечающий за: внутриигровую музыку, обновление счётчика рекорда, случайный порядок блоков, фиксацию блоков при прикосновении с нижними блоками, обновление поля

5)colors.py - код, который хранит информацию о используемых цветах

6)block.pu - это код, отвечающий за перемещение блоков

7)blocks.py - код содержащий информацию о видах блоков, о том как они поворачиваются, их координаты

8)position.py - код с координатами, для использования для обозначения позиций у блоков, в коде blocks.py
